# HMS Assignment

This repository contains microservices for a Hospital Management System (HMS):

- Doctor Service
- Patient Service
- Appointment Service
- Prescription Service
- Billing Service
- Payment Service

## Structure

Each service is located in its own folder. See the `docker-compose.yml` for service orchestration.

## Getting Started

1. Clone the repository.
2. Use Docker Compose to build and run all services:
   ```
   docker-compose up --build
   ```
3. See the `docs/` folder for API documentation and usage details.
